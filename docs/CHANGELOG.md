# Changelog - FinUnity

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2024-01-XX

### Adicionado
- Sistema completo de autenticação (registro e login)
- Dashboard com gráficos de gastos por categoria
- Gráfico de evolução mensal (receitas vs despesas)
- Sistema de transações completo (CRUD)
- Sistema de metas de poupança com barras de progresso
- Sistema de contas bancárias múltiplas
- Página de relatórios detalhados
- Exportação para CSV e PDF
- Importação de extratos CSV
- Sistema de notificações em tempo real
- Sugestões inteligentes da IA
- Sistema de regras de divisão para casais
- Página de configurações
- Central de ajuda e documentação
- Componentes reutilizáveis (LoadingSpinner, ErrorMessage)
- Auto-geração diária de sugestões da IA
- Classificação automática de transações por categoria

### Funcionalidades MVP
- ✅ Cadastro de usuários (solteiro/casal)
- ✅ Registro de receitas e despesas
- ✅ Painel com saldo geral e por conta
- ✅ Orçamentos mensais por categoria
- ✅ Metas de poupança
- ✅ Relatórios mensais
- ✅ Exportação CSV/PDF
- ✅ Assistente IA com sugestões

### Funcionalidades Avançadas
- ✅ Divisão automática de contas para casais
- ✅ Regras personalizadas de divisão
- ✅ Gráficos e visualizações
- ✅ Importação de extratos
- ✅ Notificações inteligentes
- ✅ Análise de gastos por categoria

### Melhorias Técnicas
- Sistema de rotas protegidas
- Interceptors para autenticação
- Tratamento de erros centralizado
- Validação de dados no frontend e backend
- TypeScript em todo o projeto
- Componentes modulares e reutilizáveis
